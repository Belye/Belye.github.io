Python入门之基本语法和数据类型

发表于 2019-03-13 | 分类于 Python入门 ， 基本语法和数据类型 | | 阅读次数:

注释

  • 单行注释使用 “#” 开头
  • 多行注释使用 ‘’‘注释内容’‘’

变量

  • 变量命名规则采用驼峰命名法,驼峰命名法分为大驼峰(UserName)和小驼峰(userName)

数据类型

Number

int(整数)
float(浮点数)
complex(复数)
bool(布尔)

String(字符串))

  • 在pyton中字符串采用单引号或者双引号，同时可以使用反斜杠 () 表示转义特殊字符

字符串索引和切片

str1 = 'abcdefg'
print(str1[0])
print(str1[0:2]) #左闭右开
print(str1[-1:-4:-1]) #第三个参数表示步长
print(type(str1[0]))

a
ab
gfe
<class 'str'>

字符串的加法和乘法

str1 = '---'
str2 = '***'
print(str1 + str2)
print(str2*5)

---***
***************

List(列表)

  • 列表使用方括号中间元素用逗号隔开

列表的索引和切片

list1 = ['你好','Leiy','Syy']
print(list1[:])
print(list1[0:2])
print(type(list1))
print(type(list1[0]))

['你好', 'Leiy', 'Syy']
['你好', 'Leiy']
<class 'list'>
<class 'str'>

列表拼接

list1 = ['你好']
list2 = ['苏大']
print(list1+list2)

['你好', '苏大']

修改列表元素

list1 = ['你好','abcd',123]
# list1[1][1] = 'a'  #error 字符串可以被索引但是没法子修改
list1[0] = 'a'
print(list1)

['a', 'abcd', 123]

Tuple(元祖)

  • 元祖使用小括号中间元素用逗号隔开
  • tuple的元素不可以改变但是可以包含可以改变的对象如list

元祖索引

tuple1 = ('abc', '你好', 123, [1, 2, 3, 4])
print(tuple1)       # 打印整个元祖数据
print(tuple1[0:2])  # 打印第二个元祖数据

tuple1[3][0] = 4    # 修改元祖第3个数据的第一个数据 元祖虽然不可变但列表可变
print(tuple1)       # 打印修改后的元祖

('abc', '你好', 123, [1, 2, 3, 4])
('abc', '你好')
('abc', '你好', 123, [4, 2, 3, 4])

Dictionary(字典)

  • 字典是一种映射类型,使用{}
  • 键值key必须使用不可边类型(字符串,数值,元祖),同一个字典中,键必须是唯一的
  • 创建空字典用{}

字典的索引

dict1 = {'name':'Leiy','age':17,'sex':'man'}
print(dict1)
dict2 = {'name':'Leiy','age':17,'sex':'man','name':'syy'} #同一个字典产生了两个name 后面一个name覆盖了前面
print(dict2)


print(dict2['name'])#这边是索引dict1的name值


dict3 = {'zhuang':{'age':17,'name':'Leiy'}}
print(dict3['zhuang'])        #字典索引
print(dict3['zhuang']['name'])#多层索引的获取

{'name': 'Leiy', 'age': 17, 'sex': 'man'}
{'name': 'syy', 'age': 17, 'sex': 'man'}
syy
{'age': 17, 'name': 'Leiy'}
Leiy

运算符

算术运算符

  • 算术运算符有七种分别为
    +, -, , /, %(取余)), *(幂)), //(取整)

赋值运算符

  • 赋值运算符有八种分别为
    =, +=, -=, =, /=, %=, *=, //=

比较运算符

  • 比较运算符有六种
    ==, ！=, >, <, >=, <=

逻辑运算符

and(布尔‘与’),or(布尔’或‘),not(布尔’非‘)

位运算符

&(按位与)
|(按位或)
^(按位异或)
～(按位取反)
<<(左移运算符)
>>(右移运算符)
