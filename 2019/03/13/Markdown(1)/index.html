Markdown标题设置

Markdown标题的设置通过设置 # 的个数来确定而 # 个数的范围是 1～6

    # 一级标题
    ## 二级标题
    ### 三级标题
    #### 四级标题
    ##### 五级标题
    ###### 六级标题
    ps: # 和标题中间有空格

图片和超链接

超链接

Markdown中插入超链接的格式为  [link text](URL 'title text')
如：[Baidu](http://www.baidu.com)

超链接效果展示

Baidu

图片

Markdown中插入图片于超链接类似 只不过前面多了 ！
格式为 ！[link text](URL 'title text')
建议使用sm.ms生成图片url 网站为 https://sm.ms/

如：  ![姚贝娜](https://i.loli.net/2018/08/12/5b700577810ed.jpg)

表格

表格的生成在Markdown中比较麻烦！
:--- 代表左对齐
:--: 代表居中对齐
---: 代表右对齐

| Tables        | Are           | Cool  |
| ------------- |:-------------:| -----:|
| col 3 is      | right-aligned | $1600 |
| col 2 is      | centered      |   $12 |
| zebra stripes | are neat      |    $1 |
