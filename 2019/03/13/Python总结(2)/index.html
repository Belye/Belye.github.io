输入与输出

inpute

  • 通过键盘获取数据，然后保存到指定变量中，都是按照字符串方式进行保存
a = input('请输入数据1:')
b = input('请输入数据2:')
print(a + b)  # a,b都是按照字符串形式保存

a = int(input('请输入数据1:'))
b = int(input('请输入数据2:'))
print(a + b)  # a,b强转int类型

请输入数据2:2

通用格式

  • print函数中参数sep指的是当打印各个值时,各个值之间的分割方式,默认空格,end参数表示print中传入值输出完成后结束符号,默认换行
a = 1
b = 2
c = 3
print(a,b,c)
print(a,b,c,end = 'love')
print()
print(a,b,c,sep = '|')

a = 1
b = 2
print('{}输出{}'.format(a,b))
print('{1}输出{0}'.format(a,b)) #交换索引位置
print('{a}输出{b}{a}'.format(a = 1,b = 2))

常见字符串函数

find

  • 检查字符串是否包含某个指定字符
str1 = 'Hello World'
#find
print(str1.find('o'))#只是返回第一个所要寻找字母的所在地点
print(str1.find('a'))#找不到会返回 -1
print(str1.find('lo'))#可以寻找一个整体
print(str1.find('r',0,4))#限定了寻找范围的find

4
-1
3
-1

index

  • 检查字符串是否包含某指定字符
str1 = 'Hello World'
#index
print(str1.index('r'))#index与find不同的是当找不到的时候会报错

8

count

  • 统计某个字符在指定范围内出现的次数
str1 = 'Hello World'
#count
print(str1.count('o'))#2
print(str1.count('o',0,6))#同理限定范围内的寻找并计数 前闭后开

2
1

replace

  • 将str1中的str1替换为str2,如果指定count,则不超过count次
str1 = 'hello world hello china'
str2 = str1.replace('hello','Hello',1)#这边只是替换第一个 默认是全部替换

print(str1.replace('hello','Hello'))#默认把所有的都替换掉
print(str1)#本质上str1无法替换 字符串无法修改只是会生成新的字符串
print(str2)#替换后

Hello world Hello china
hello world hello china
Hello world hello china

split

  • 分割字符,如果maxsplit有指定值，则仅分割maxsplit个字符串
str1 = 'hello world hello china'
print(str1.split(' ')) #空格地方分割
print(str1.split('o',3)) #在小o地方分割最多分割3处

['hello', 'world', 'hello', 'china']
['hell', ' world hello china']

capitalize

  • 将字符串首字母大写
str1 = 'hello world hello china'
print(str1.capitalize()) #第一个字母大写

Hello world hello china

title

  • 将字符串中每个单词的首字母大写
str1 = 'hello world hello china'
print(str1.title())

Hello World Hello China

startwith & endswith

  • 检查字符串是否以obj开头,是则返回True,否则为False
  • 检查字符串是否以obj结尾,是则返回True,否则为False
str1 = 'hello world hello china'
print(str1.startswith('hello'))
print(str1.endswith('hello'))

True
False

lower & upper

  • 将字符串转换为小写
  • 将字符串转换为大写
str1 = 'hello world hello china'
print(str1.lower())
print(str1.upper())

hello world hello china
HELLO WORLD HELLO CHINA

ljust & rjust & center

  • 返回一个原字符串左对齐,并使用空格填充至长度width的新字符串
  • 返回一个原字符串右对齐,并使用空格填充至长度width的新字符串
  • 返回一个原字符串居中,并使用空格填充至长度 width 的新字符串
str1 = 'hello'
print(str1.ljust(10))  #开辟10个位置 并且左对齐
print(str1.rjust(10))  #开辟10个位置 并且右对齐
print(str1.center(10)) #开辟10个位置 并且中间对齐

hello     
    hello
hello

lstrip & rstrip & strip

  • 去除字符串左边空白字符
  • 去除字符串右边空白字符
  • 去除字符串两边空白字符
str1 = '     Hello     ' 
print(str1.lstrip())#默认删除左空格 
print(str1.lstrip('%'))#这句话是说删除%  这里没加
print(str1.strip())#这句话是从两边删除

Hello     
    Hello     
Hello

partition

  • 可以将字符串按照str1分割成三个部分 str1前,str1, str1后
str1 = 'hello world hello china'
print(str1.partition('world')) #将字符串分割成三个部分 前 world 后 返回为元祖

('hello ', 'world', ' hello china')

join

  • str1 中每个字符串后面插入str1,构造出一个新的字符串
str1 = '-'
list1 = ['hello','world','hello','china']
print(str1.join(list1)) #默认将,号转换为下划线  将列表转换为字符串

hello-world-hello-china

issapce & isalnum

  • 如果str1中只包含空格,返回True,否则为False
  • 如果str1中只包含数字,返回True,否则为False
str1 = 'asd123 321asd'
print(str1.isspace())#如果str1中只包含空格则返回true 否则false
print(str1.isalnum())#如果都是字母或数字则返回true,否则false

False
False

isdigit & isalpha

  • 如果str1中只包含数字,返回True,否则为False
  • 如果str1中只包含字母,返回True,否则为False
str1 = 'asd123 321asd'
print(str1.isdigit())#如果都是数字则返回True否则false
print(str1.isalpha())#判断是否全是字母

False
False
